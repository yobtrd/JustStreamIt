# JustStreamIt

Interface utilisateur désignée pour une application web permettant de visualiser en temps réel un classement de films.


## Fonctionnalités

- Interface dynamique 
- Responsive Design

## Usage

L'UI est destinée à fonctionner via une API spéficiquement désignée pour elle.

## Maintenabilité du code

- Respect des normes W3C.
- Respect des pratiques JS.
